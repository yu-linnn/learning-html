CSS Box Model

All of the text written are in the content box.

The CSS box comprises (from outside to inside):

  • Margin
  • Border
  • Padding
  • Content

Margin, Border and Padding are transparent unless specified.

Background-color only applies to padding and content.

We can combine padding properties into one (Padding:20px,20px,20px,20px).

No margin: margin:0

Now let's try it!

{background-color:lightgrey; padding:20px;border-color:black;border-width:2px;border style:solid}

Padding is the space between the content and the border.
See the space between the border and the sides of the webpage? That's the margin!

{margin-top/left/bottom/right:--px}
{padding-left/right/top/bottom:--px}

Overflow Property

overflow:

  • scroll
  • hidden
  • auto
  • clip
  • visible

visible: The overflow is not clipped. It renders outside the element's box. This is default

hidden: The overflow is clipped, and the rest of the content will be invisible. Content can be scrolled programmatically (e.g. by setting scrollLeft or scrollTo())

clip : The overflow is clipped, and the rest of the content will be invisible. Forbids scrolling, including programmatic scrolling.

scroll: The overflow is clipped, but a scroll-bar is added to see the rest of the content

auto: If overflow is clipped, a scroll-bar should be added to see the rest of the content

inset: Sets this property to its default value. Read about initial
inherit: :Inherits this property from its parent element. Read about inherit
